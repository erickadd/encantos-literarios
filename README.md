Landing page responsiva para o clube de assinatura literário "Encantos Literários", desenvolvida com HTML5 e CSS3 puros. O projeto apresenta animações suaves, design atraente e foco em conversão.

## 🎯 Sobre o Projeto

Este projeto foi desenvolvido como um desafio para criar uma página de vendas para um clube de assinatura de livros. O objetivo é apresentar os benefícios do clube de forma envolvente, destacando o kit mensal, os planos disponíveis e incentivar a assinatura.

## ✨ Funcionalidades

- **Design Responsivo** - Adapta-se perfeitamente a diferentes tamanhos de tela (mobile, tablet, desktop)
- **Header Fixo** - Navegação sempre visível com logo e CTA
- **Seção Hero** - Apresentação principal com título chamativo e ilustração animada
- **Kit do Mês** - Exibição interativa dos itens inclusos com animações sequenciais
- **Cards de Planos** - 3 opções de assinatura com destaque para o plano semestral
- **Footer Completo** - Links úteis e redes sociais
- **Animações CSS** - Transições suaves, hover effects e microinterações

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Estilização avançada com:
  - Flexbox e Grid para layouts
  - Animações keyframes
  - Transitions e transforms
  - Variáveis CSS
  - Media queries para responsividade
- **Font Awesome** - Ícones para redes sociais e benefícios

📱 Responsividade
O projeto é totalmente responsivo com breakpoints:

Desktop > 1024px - Layout completo

Tablet 768px - 1023px - Ajustes no espaçamento

Mobile < 768px - Layout empilhado com menu adaptado

🎭 Animações Implementadas
Elemento	Animação
Header	Slide down na entrada
Livro	Flutuação e abertura das páginas
Estrelas	Brilho (twinkle)
Features do Kit	Aparecem sequencialmente
Cards	Escala e hover effects
Badge	Pulsação

🔧 Personalização
Para alterar as cores
Edite as variáveis CSS no início do arquivo style.css:
:root {
    --primary-color: #8b4513;
    --secondary-color: #a0522d;
    --dark-color: #2c1810;
    --light-color: #fdf5e6;
}

Para substituir as imagens
Adicione suas imagens na pasta images/

Atualize os caminhos no arquivo index.html
